README
При запуске игры нажмите кнопку "Start Game" левой кнопкой мыши и запустится первый уровень.
Для управления персонажем используйте кнопки "A", "D" или стрелку влево и стрелку вправо.
Для прыжка используйте пробел. Для увеличения скорости передвижения персонажа при передвижении используйте кнопку "Shift".
Для получения большего итогового счета, собирайте как можно больше монет, они разбросаны по всему уровню. При падении на шипы, вы проигрываете и можете начать уровень с начала или выйти из игры.
После прохождения уровня, вы можете запустить следующий или выйти из игры.
После прохождения всех трех уровней, будет доступна только кнопка выйти.
